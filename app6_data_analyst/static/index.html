3 QA Test Scenarios

Happy Path
Basic Retrieval
Ask "Which category has the highest number?" Expect: agent calls retrieve_data, finds the answer, cites a page number.
Computation
Python Calculation
Ask "What is the total sum of all figures?" Expect: agent calls run_python to compute the sum — check the code + output in agent steps.
Multi-Turn Memory
Follow-Up Question
Ask two related questions in sequence. The second should reference the first answer. Expect: contextual, consistent answers without repeating retrieval.
Memory Tool
Recall Previous Answer
After asking 2+ questions, ask "Remind me what the highest category was." Expect: agent calls recall_previous_answer tool, returns prior answer.
Reset Test
Clear Conversation
Ask a question, note the answer, then click "Clear Conversation" and ask "What was the total we calculated earlier?" Expect: agent has no memory of it.
Out-of-Scope
Unanswerable Question
Ask "What information is NOT present in this document?" Expect: agent acknowledges it can only answer from retrieved content — not fabricate missing data.

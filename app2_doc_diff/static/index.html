3 QA Test Scenarios

Happy Path
Near-Identical Docs
Upload two versions of the same policy with 1–2 small edits. Expect: Minor severity, few modified items, high overlap.
Major Change
Heavily Revised Doc
Upload an original doc and a significantly rewritten version. Expect: Major severity, many added/removed/modified items.
Edge Case
Same Doc Both Slots
Upload the exact same PDF as both A and B. Expect: Minor severity, 0 or near-zero changes, "nothing changed" summary.
Cross-Domain
Completely Different Docs
Upload two unrelated PDFs (e.g. a resume vs a policy). Expect: many removals/additions, near-zero modified, confusing summary.
Metadata Check
Verify PDF Metadata
After upload, inspect the metadata box: page count, word count, file size, author/title fields. Verify they match the actual PDF properties.
Error Handling
Analyze Without Upload
Click Analyze without uploading both files. Expect: clear error message, button stays disabled until both docs loaded.
